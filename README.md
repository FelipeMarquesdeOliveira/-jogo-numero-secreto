# Jogo do Número Secreto

Este é um jogo simples desenvolvido como parte do curso de Lógica de Programação com JavaScript da Alura. O jogo consiste em adivinhar um número secreto gerado aleatoriamente.

## Funcionalidades

- **Gerar Número Aleatório:** O jogo gera um número aleatório que o jogador deve tentar adivinhar.
- **Verificar Chute:** O jogador insere um número e o jogo informa se o número é maior ou menor que o número secreto.
- **Reiniciar Jogo:** Após acertar o número secreto, o jogador pode reiniciar o jogo para tentar novamente.

## Como Jogar

1. Abra o link: [JOGO](https://jogo-lake-zeta.vercel.app/)
2. Uma mensagem inicial será exibida indicando que você deve escolher um número entre 1 e o máximo definido.
3. Insira um número no campo de input e pressione Enter para verificar se você acertou.
4. O jogo informará se o número é maior ou menor que o número secreto.
5. Continue tentando até acertar o número secreto.
6. Após acertar, o botão "Reiniciar Jogo" estará disponível para jogar novamente.
